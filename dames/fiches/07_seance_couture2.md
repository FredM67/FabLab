# Séance 7 — Couture pochette, partie 2 (45 min)

## Objectifs

- Assembler les deux faces du sac
- Faire la coulisse
- Passer le cordon

## Déroulement

### Rappel et reprise (5 min)

- Reprendre les pièces de la séance 6
- On a : 1 panneau avec appliqué + 1 panneau avec poche
- Aujourd'hui on assemble le tout

### Assembler les 2 faces (15 min)

1. Poser les 2 panneaux ENDROIT contre ENDROIT
   (la poche et l'appliqué sont à l'intérieur)
2. Épingler les 3 côtés : gauche, bas, droite
3. Coudre à 10 mm du bord les 3 côtés
4. **IMPORTANT** : en haut de chaque côté, s'arrêter
   à 25 mm du bord supérieur (laisser l'ouverture
   pour la coulisse). Point d'arrêt.

```
    ← 25mm non cousu →|
    |                    |
    |  coudre ici        |
    |  (10mm du bord)    |
    |                    |
    +----coudre ici------+
```

### Faire la coulisse (15 min)

1. Retourner le sac (l'endroit apparaît)
2. En haut, plier le bord vers l'intérieur :
   - 1ᵉʳ pli : 10 mm
   - 2ᵉ pli : 20 mm (forme le tunnel pour le cordon)
3. Repasser pour maintenir les plis
4. Épingler
5. Coudre à 2 mm du bord inférieur du pli
   (faire tout le tour de l'ouverture)

### Passer le cordon (5 min)

1. Épingle à nourrice sur le bout du cordon
2. Enfiler dans la coulisse par une ouverture latérale
3. Faire le tour complet, ressortir par la même ouverture
4. Nouer les 2 bouts

### Test (5 min)

- Tirer le cordon : le sac se ferme bien ?
- Y mettre quelques pièces de puzzle pour vérifier la taille
- Couper les fils qui dépassent

## Schéma récapitulatif

```
Étape 1 : endroit     Étape 2 : retourner
contre endroit         + plier la coulisse

+--ouverture--+        +--ouverture--+
|##ouverture##|        |  [coulisse] |
|             |        |             |
| (appliqué   |  →→→   |  sac fini   |
|  et poche   |        |  endroit    |
|  dedans)    |        |  visible    |
|             |        |             |
+---cousu-----+        +-------------+
```

## Matériel

- Pièces cousues de la séance 6
- Machines à coudre
- Cordon ~60 cm par pochette
- Épingles à nourrice
- Fer à repasser
- Épingles, ciseaux, fil
